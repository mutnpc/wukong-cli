# Wukong CLI

Wukong CLI is a powerful terminal tool that helps you interact with [Wukong](https://wukong.today) directly from your command line.

Sync your data, manage your workflow, and stay productive â€” all without leaving your terminal.

---

## Quick Install

```bash
npm install -g @wukong.today/wukong-cli
```

## Quick Start

```bash
# Login to your account
wukong login

# Sync your data
wukong sync --pull

# Start interactive mode
wukong
```
