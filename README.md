# Wukong CLI

Official terminal tool for Wukong.

Website: https://wukong.today

## Install

```bash
npm install -g @wukong.today/wukong-cli
```

## Usage

```bash
wukong
wukong login
wukong sync --pull
```

## Help

```bash
wukong --help
```
