# Wukong CLI

Wukong CLI is the official terminal tool for Wukong.

## Install

```bash
npm install -g @wukong.today/wukong-cli
```

## Get started

```bash
wukong
wukong login
wukong sync --pull
```

## Help

```bash
wukong --help
```
