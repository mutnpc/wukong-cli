# Wukong CLI ![](https://img.shields.io/badge/Node.js-18%2B-brightgreen?style=flat-square) [![npm]](https://www.npmjs.com/package/@wukong.today/wukong-cli)

[npm]: https://img.shields.io/npm/v/@wukong.today/wukong-cli.svg?style=flat-square

Wukong CLI is a private, proprietary terminal companion for Wukong. This repository is **issue tracking only** and does not contain source code.

## Get started

1. Install the CLI:

```bash
npm install -g @wukong.today/wukong-cli
```

2. Run Wukong:

```bash
wukong
```

3. Log in and sync:

```bash
wukong login
wukong sync --pull
```
