# Contributing to Wukong CLI

Thank you for your interest in contributing to Wukong CLI!

Since Wukong CLI is a closed-source project, we do not accept code contributions. However, we greatly value community feedback and participation in other ways.

## How You Can Contribute

### üêõ Report Bugs

Found a bug? Help us improve by reporting it:

1. Use the built-in bug reporter: `wukong bug`
2. Or file a [GitHub issue](https://github.com/mutnpc/wukong-cli/issues/new?template=bug_report.md)

**Please include:**
- Your operating system and version
- Node.js version (`node --version`)
- Wukong CLI version (`wukong --version`)
- Steps to reproduce the issue
- Expected vs. actual behavior
- Relevant logs or screenshots

### üí° Request Features

Have an idea for a new feature?

1. Check [existing feature requests](https://github.com/mutnpc/wukong-cli/issues?q=label%3Aenhancement) first
2. Open a new [feature request](https://github.com/mutnpc/wukong-cli/issues/new?template=feature_request.md)

### üìñ Improve Documentation

Found a typo or unclear documentation? Let us know by:

1. Opening an issue describing the problem
2. Suggesting the improvement

### üåç Help with Translations

We welcome help translating Wukong CLI to other languages. Contact us at [i18n@wukong.today](mailto:i18n@wukong.today).

## Code of Conduct

Please be respectful and constructive in all interactions. We are committed to providing a welcoming and inclusive environment for everyone.

- Be kind and courteous
- Respect differing viewpoints
- Accept constructive criticism gracefully
- Focus on what is best for the community
